Learning the Code

When I first started, I had to learn how to structure a webpage, link different files together, and style everything so it looked modern and clean. One of the biggest challenges was getting the CSS Grid layout to work perfectly for my project cards so they would line up neatly and be easily searchable using JavaScript.

Going Live

Beyond just writing the code, I also learned how to use GitHub Pages to actually host the website on the internet. It was an amazing feeling to upload my files—like my index and style sheets—and suddenly see my work live for anyone to visit.

Conclusion

This project taught me patience, problem-solving, and a lot of technical skills. Whenever something broke, like an image not loading or a text block overflowing, I had to dig into the code and figure out why. Overall, creating this portfolio was incredibly rewarding, and I am very proud of how it turned out.
